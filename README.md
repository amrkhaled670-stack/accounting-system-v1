# 🚀 Accounting System v1
**نظام محاسبي متكامل تم تطويره بواسطة عمرو خالد**

## 🌟 أبرز المميزات (Features)
* 🔐 **نظام تسجيل دخول:** تأمين البيانات وحمايتها.
* 📊 **لوحة تحكم (Dashboard):** عرض سريع للمبيعات والمنتجات.
* 🌍 **دعم كامل للغة العربية:** واجهة RTL متوافقة مع المستخدم العربي.
* 🌙 **الوضع الليلي (Dark Mode):** تصميم أنيق ومريح للعين.
* 📦 **إدارة المخازن:** تتبع الموردين والمخزون وحركة الفواتير.

## 🛠 التقنيات (Tech Stack)
* **Framework:** Next.js 14
* **Database:** Prisma ORM (SQLite/PostgreSQL)
* **UI:** Tailwind CSS & Shadcn UI
* **Fonts:** Cairo (Arabic Font Support)
